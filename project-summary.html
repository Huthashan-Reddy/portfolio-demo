Aarav Sharma Student Portfolio

Interactive React + TypeScript Web Application

📋 Project Overview

A modern, fully responsive student portfolio website showcasing academics, projects, skills, certifications, and testimonials. Built with React 19, TypeScript, and Vite, featuring advanced animations powered by GSAP, Three.js, and interactive 3D elements.

Project Goal

Create an engaging, visually stunning personal portfolio that demonstrates technical expertise through a combination of modern web technologies, complex animations, and interactive user experiences.

🔍 Technical Decisions & Rationale

Why React 19 + TypeScript?

React 19 provides the latest features and performance improvements, while TypeScript ensures type safety across a complex component hierarchy. This combination improves developer productivity and reduces runtime errors.

Why Vite?

Vite offers exceptional build performance with instant HMR, essential for rapid development iteration. The dev server startup is near-instantaneous, improving developer experience significantly.

Why GSAP for Animations?

GSAP is the industry-standard animation library with superior performance, ease of use, and ScrollTrigger plugin for scroll-linked animations. The learning curve is minimal for developers familiar with animation libraries.

Why Three.js + React Three Fiber?

Three.js provides robust 3D graphics capabilities, while React Three Fiber seamlessly integrates Three.js with React's component model. This approach provides better code organization and reusability compared to vanilla Three.js.

Why Tailwind CSS?

Tailwind's utility-first approach enables rapid development with consistency. Custom configuration supports the design system, and the small bundle size keeps the application lightweight.
